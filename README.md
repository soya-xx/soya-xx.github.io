# 向向Soya · Official Site

官方概念网站 — **一个拥有两副面孔的网站**，正如她本人。

> 向向Soya，诞生于时空裂缝的布偶猫系兽耳猫娘。
> 金发蓝瞳，女仆风裙装，穿梭于无尽时间线，只为寻找那个唯一不变的真名。
> ——以及，封存在她灵魂深处的「第二人格」。

## 运行

纯静态站点，无构建步骤：

```bash
python3 -m http.server 8765
# open http://localhost:8765
```

（GSAP 与字体走 CDN；离线打开时优雅降级为无动画模式。）

## 表 / OMOTE — 你看到的她

- **加载页**：摇晃的铃铛 + 时间线连接进度
- **首页**：立绘呼吸动画、鼠标视差、巨型描边字、格纹背景、跳动的 REC 时间码
- **档案**：取景器相框（实时时间码）、档案数据表、损坏的统计数字（永远在闪烁的「找到爱人的次数」）
- **物语**：四章滚动时间轴，讲述表层设定
- **表情差分**：从设定图自动裁切的四枚表情卡（3D 倾斜悬停）
- **画廊**：瀑布流 + 灯箱（键盘 ←/→/Esc）
- **直播**：周程表 + 关注按钮
- 飘落的花瓣与星星粒子、胶片噪点、自定义光标、文字乱码解码动画
- **音乐**：右上角 ♪ —— 用 WebAudio 实时合成的八音盒摇篮曲（无任何音频文件）

## 里 / URA — 找到你了♡

## 技术

单页静态站：手写 HTML/CSS/JS（零框架），GSAP + ScrollTrigger 做滚动叙事，
Canvas 粒子引擎（表里双模式），WebAudio 程序化合成 BGM/铃声/心跳，
CSS 变量驱动的整站双主题，`prefers-reduced-motion` 与移动端完整适配。

图片资产由 Python/PIL 流水线处理：透明边裁切、杂点清理、WebP 压缩（19MB → ~1.9MB）、
从设定图程序化检测并裁切表情差分、生成双态 favicon（表/里）。
